Firstly please check if you set up correctly in the App, according to video manual.Secondly the alarm system need be armed in advance. Thirdly the device need connect the internetyou can check if the internet LED indicator is lighting in blue. Forth check if you have alarm SMS & call quota. Finally once you trigger alarm and get SMS & call, in the coming 10 minutes, the alarm system will not release a new one.
When the host is disarmed, I trigger a device, the host starts to alarm and I got alarm call, thats the matter?
Please check if you added this device to 24H alarm device list, such kind of device like smoke detector will trigger alarm even though the host is disarmed.
